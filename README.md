# 🐍 Sistema de Gestão de Itens - CRUD Python

Este projeto é um software desktop completo que demonstra a aplicação de **Estruturas de Dados** e **Persistência Relacional**. Desenvolvido com uma interface moderna e intuitiva.

## 🚀 Funcionalidades
- **Interface Gráfica (GUI):** Visual moderno com suporte a temas.
- **Banco de Dados:** Persistência de dados local utilizando SQLite.
- **Operações CRUD:** 
  - Cadastro de novos itens (Create).
  - Listagem em tempo real (Read).
  - Remoção de registros via ID (Delete).
- **Executável:** Pronto para distribuição no Windows (.exe).

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Python 3.12+
- **Interface:** CustomTkinter
- **Banco de Dados (ORM):** SQLAlchemy
- **Ambiente Virtual:** venv
- **Distribuição:** PyInstaller

## 📦 Como rodar o projeto
1. Clone o repositório:
2. pip install -r requirements.txt
python app_final.py
